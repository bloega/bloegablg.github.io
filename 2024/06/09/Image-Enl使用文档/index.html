Image Enl使用文档

Image Enl是一款基于深度学习算法，对图片进行无损放大的图片处理工具

背景

对于当下人们对于图片清晰度的需求，无论人们从网络还是相册中的图片，都会面对图片不清晰或者放大后失真（图像失真指数码图像与真实拍摄环境的差异）的情况，即便网络上已有一些工具来解决此问题，但都有着收费，效果不明显，国内无法访问等问题，所以，ImageEnl应实势而生

如何工作

技术栈：

  1. Python

  2. Numpy

  3. Pillow

  4. Shutil

使用Pillow打开图片，打开后经算法将图片放大后写入到模板文件（tmp.png）然后将图片写入到程序自动创建的图片中，达到图片变清晰的效果

运行效果

本用户界面使用tkiner进行设计，并且进行了美化，使其更加适合现代人群

按钮名称解释
选择图片选择要放大的图片路径（注：文件路径中不能有中文）
选择保存路径选择要保存文件的路径（注：文件路径中不能有中文）
选择放大倍数选择放大倍数（2x为2倍，4x为4倍…）
开始放大开始对图片进行放大（放大速度与电脑性能有关）

选择图片后，图片会在界面底部的画框中显示类似于这样

放大后会打开图片

优势

  1. 使用Numpy对文件OI（文件打开与输入）操作进行加速，尽量压缩到10分钟以内
  2. UI设计更加贴合现代化人群
  3. 采用离线操作，不依赖任何网络文件
  4. 纯Python编写且采用多种Python modules使代码简洁，易懂

劣势

  1. 由于运行在本地，只能使用本地资源进行放大操作无法使用cuda（显卡加速）
  2. 只能放大png文件（技术有限）
